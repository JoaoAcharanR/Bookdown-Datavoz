3 Colapsar variables

En algunos casos tendremos baterías de preguntas que tendrán los valores en columnas independientes,

# creamos un vector que contenga los nombres de nuestras variables
# en este caso las variables p1, p2, p3
variables <- colnames(df[2:4])

# para ejecutar correctamente el codigo debemos insertar el vector que hemos creado
# anteriormente, dentro de la funcion syms(VECTOR)
df <- df %>% 
  mutate(estado_civil = coalesce(!!! syms(variables)))
df %>% 
  flextable()
